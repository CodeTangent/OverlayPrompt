# OverlayPrompt
Aplicação simples em Tkinter que exibe um aviso transparente sempre no topo da tela, sobrepondo qualquer aplicação aberta. A mensagem funciona como um lembrete visual rápido e pode ser fechada com um clique. O projeto foi empacotado em um standalone, eliminando a necessidade de instalação do Python ou dependências, tornando-o completamente portátil
